# Candidate List Viewer

A **React** application to view a list of candidates, search by name or skills, and sort them by years of experience. The app fetches data from a **backend API**, which provides candidate information.

---

## ✨ Features

- **Search**: Filter candidates by name or skills.
- **Sort**: Sort candidates by years of experience in **ascending** or **descending** order.

## 💻 Installation

### Prerequisites

Before starting, make sure you have the following installed on your system:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) (npm is included with Node.js)

### Steps to Run the Project Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/YourUsername/Task1.git

2. **Navigate into the project folder:**

   ```bash
   cd Task1
   
3. **Install frontend dependencies:**

    ```bash

      cd client
      npm install

4. **Run the frontend application:**

   ```bash

     npm start
   
This will start the frontend React app at http://localhost:3000.

5. **To deploy the app to GitHub Pages, run the following commands:**

    ```bash

      npm run build
      npm run deploy
